烹飪必備地辛香料之一「大蒜」，剝皮方法有許多種，有人將其壓扁去皮，也有人慢慢剝，但到底該如何快速又方便去皮呢？最近推特上就有一段影片，利用小技巧約莫2至3秒就能做好剝大蒜，讓網友嘖嘖稱奇。   原PO「@VPestilenZ」在推特上傳一段25秒影片，分享如何去除大蒜外皮，可見他不用複雜的工具，只要一把小刀就能輕鬆搞定。廚師左手拿著一顆大蒜，右手握著一把小刀直接插入大蒜，輕鬆一扭，一眨眼的工夫就能取出「光溜溜大蒜」，毫不費力，手也不會因為碰到大蒜而殘留氣味。   快速便捷的方法不到一星期就有40萬人按讚、13萬人分享及1千910萬觀看次數，網友紛紛直呼「太神奇啦！」、「我試了一下，沒影片快，但真的比之前方法有效多了」、「我學會了」、「大開眼界」。
